# WoW TBC Class Colors & Spec Icons Reference

## Class Colors (text color-coding)

These are the standard WoW class colors used in the Discord signup tool. When a player name is displayed in colored text, the color identifies their class.

| Class   | Color                  | Hex       | Description                    |
|---------|------------------------|-----------|--------------------------------|
| Warrior | Tan / Light Brown      | #C69B6D   | Sandy brown, easy to confuse with background |
| Paladin | Pink                   | #F48CBA   | Bright pink / rose             |
| Hunter  | Green                  | #AAD372   | Light green / lime             |
| Rogue   | Yellow                 | #FFF468   | Bright yellow                  |
| Priest  | White                  | #FFFFFF   | White (can be hard to see on light backgrounds) |
| Shaman  | Blue                   | #0070DD   | Medium blue                    |
| Mage    | Light Blue / Cyan      | #3FC7EB   | Cyan / sky blue                |
| Warlock | Purple / Lavender      | #8788EE   | Light purple / periwinkle      |
| Druid   | Orange                 | #FF7C0A   | Bright orange                  |

### Commonly confused pairs
- **Shaman (blue)** vs **Mage (cyan)** — Shaman is darker blue, Mage is lighter/more cyan
- **Warrior (tan)** vs **Paladin (pink)** — Warrior is brownish, Paladin is distinctly pink
- **Warlock (purple)** vs **Shaman (blue)** — Warlock is more purple/lavender, Shaman is pure blue

## Notes

- In the Discord signup tool, the **category header** (e.g., "Tank", "Warrior", "Druid") groups by class or role.
- The **spec icon** next to each player name shows their chosen specialization for that raid.
- Players signing up under "Tank" as a role category may be any tanking class (Warrior, Paladin, Druid).
- **Primary identification method:** Cross-reference player names against `rules/04-player-specs.md` for class. But **always check the spec icon** — it shows the player's chosen spec for THIS raid and **takes precedence** over any previously recorded role. Players can and do change specs between raids.
- When unsure, open the reference image file (e.g., `Read icons/specs/shaman_enhancement.jpg`) and compare side by side.
- If still uncertain about a new player's class or spec, ask the user.
